Welcome to the Koinx Backend Server!

API Endpoints:

  • Get Transactions: GET user/transactions/:address
    Fetches all transactions for a specified blockchain address.
  • Get Balance and Price: GET user/balance/:address
    Fetches the balance and current Ethereum price in INR for a specified blockchain address.
